Javascript was created by Netscape employee Brendan Eich in 1995.
It was inspired by direct competition between Microsoft and Netscape and
its ally, Sun Microsystems.

Java was very popular and Eich's language was meant to shipped as a
companion language to it. Its name was part of a marketing strategy.
Originally is was called Mocha. Yeah, it lacks je ne sais quoi. Javascript
though, now that's HOT!

Now according to the source I am getting my information from, Javascript is
one of the most hated languages on the planet (article written in 2015). Yet
it is the standard programming language of the web.
