PHP is an html embedded scripting language, and we had a taste of how convenient
it can be in Dr. Lara's class. It is used for creating beautiful, precious and
dynamic web applications.

PHP's syntax is a derivative of C, Java, and Perl.

It is open source, and runs on most OSs and webservers. PHP was an acronym
for "Personal Home Page", but later some cheeky fellow recoined its meaning to
"HyperText Preprocessor". It's recursive, the acronym is its own defintion!
We'll be learning a lot more about this in class.
