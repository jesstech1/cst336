Main The First Operating System MS-DOS Windows
Windows is a modern OS of some note, developed by a modest software concern
operating out of Redmond, Washington. It has seen its share of popularity
since the mid 80s, and evolved several times since its inception.

It grew in popularity as its developers had the foresight to get on board
the GUI bandwagon. It relied heavily on mouse input before the mouse
became a properly accepted input device. In a decent display of ingenuity,
the developers shipped a game with the OS the educate customers in proper
mouse use - it was called Reversi.

Windows is still popular among a percentage of PC owners.
