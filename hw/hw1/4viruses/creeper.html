The Creeper program was written in the 70s by BBN employee, Bob Thomas.
It was a proof of concept effort. Thomas wanted to see if it was
possible to create a self replicating program. The creeper would
remove it self from previosuly infected hosts.

All it did was output a single message, "I'M the CREEPER. CATCH ME
IF YOU CAN!"

Let's all thank Mr. Thomas for his contribution to the virus development
industry. **APPLAUSE**
